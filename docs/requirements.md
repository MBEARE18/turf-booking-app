# Turf Booking App - Requirements

## App Overview
A web application for booking sports turfs (e.g., football, cricket) on an hourly basis.

## User Roles
1. **Player (Customer)**
   - Browse available slots for a specific date.
   - Book a slot.
   - Make online payments.
   - View booking history.

2. **Admin (Turf Owner)**
   - Login to the dashboard.
   - Create and manage hourly slots for specific dates.
   - Set pricing for slots.
   - Block slots for maintenance or private events.
   - View all bookings.

## Key Features
- **Booking Unit:** Hourly slots (e.g., 6:00 AM - 7:00 AM).
- **Payment:** Integrated online payment (Razorpay).
- **Scope:** Single turf management (MVP).
- **Real-time Availability:** Prevent double bookings with temporary slot locking.
